Instalace:

dbGateway
1)Spustit script dbGateway.sql


Gateway
1) Je potřeba spustit Gateway.sln jako Administrator
2) GatewayRest -> Properties -> Debug -> App URL -> mastavit IP adresu lokálního PC, nechat port 3000
3) Spustit ISS Express


Android
1) Spusti kód
2) Povolit oprávění SMS a nastavit jako defaultní SMS aplikaci
3) Otevřít aplikaci -> zadat IP serveru -> START SERVICE

	*START SERVICE - spouští službu
	*LOGOUT - odhlašuje zařízení 
	*REGISTER - registruje zařízení
	*STATUS - oznamuje stav spojení se serverem
